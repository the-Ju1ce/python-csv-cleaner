# Python CSV Cleaner

## What it does
This project loads a messy CSV file, removes blank rows and duplicate rows, standardizes column names, and exports a cleaned CSV.

## Tools used
- Python
- pandas

## Files
- `cleaner.py` — main script
- `raw_data.csv` — sample messy input
- `cleaned_data.csv` — cleaned output

## How to run
```bash
python3 cleaner.py
